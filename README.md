- **프로젝트 개요:**  
우리 팀은 스터디를 더욱 효과적으로 구성하고 관리할 수 있는 스터디 매칭 어플리케이션을 개발하는 프로젝트를 진행하였다. 이 어플리케이션을 통해 학생들은 자신의 관심 분야에 맞는 스터디 그룹을 찾거나, 자신이 원하는 주제로 스터디 그룹을 만들어 다른 학생들과 협력하며 학습할 수 있다.
